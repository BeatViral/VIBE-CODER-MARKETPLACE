# Vibe Coder Marketplace

A premium React/Vite prototype for an experience-first AI builder marketplace.

## Run locally

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```

The Vite base path is configured for GitHub Pages at `/VIBE-CODER-MARKETPLACE/`.

## Brand assets

The favicon/app icon/social banner package lives in `public/assets`.

Regenerate it with:

```bash
python scripts/generate_brand_assets.py
```

See `public/assets/brand-package.md` for the full asset list.
